# Cero-Mata-Cero-Online

Project description: An online implementation of Tic Tac Toe (Cero Mata Cero), allowing multiple players to play against each other. Developed using React, Vite, Firebase as the backend, and various complementary libraries.

## Features

- **Multiplayer Game:** Allows multiple players to compete against each other in real-time.
- **Attractive Interface:** Intuitive and user-friendly design.
- **Utilizes React and Vite:** Takes advantage of React's capabilities for the interface and Vite for fast project building.
- **Firebase Backend:** Stores data and manages game logic using Firebase.

---

## Technologies and Libraries Used

- React
- Vite
- Firebase (as backend)
- Redux-toolkit (for global state management)
- canvas-confetti (Confetti when a player wins)
- ldrs (Loader for WaitingPage)
- nanoid (Random code generation)
